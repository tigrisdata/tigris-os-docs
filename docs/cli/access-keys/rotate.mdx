# tigris access-keys rotate

Rotate an access key's secret. The current secret is immediately invalidated and
a new one is returned.

:::warning
The new secret is shown only once. Store it immediately — it cannot be retrieved again.
:::
**Alias:** `r`

## Usage

```bash
tigris access-keys rotate <id> [flags]
t3 keys r <id> [flags]
```

## Flags

| Name      | Required | Default | Description                                 |
| --------- | -------- | ------- | ------------------------------------------- |
| `--yes`, `--force` | No       | —       | Skip confirmation prompt |
```
