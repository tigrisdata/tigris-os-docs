# tigris credentials test

Verify that current credentials are valid. Optionally checks access to a
specific bucket.

**Alias:** `t`

## Usage

```bash
tigris credentials test [flags]
t3 creds t [flags]
```

## Flags

| Name             | Required | Default | Description                                   |
| ---------------- | -------- | ------- | --------------------------------------------- |
| `--bucket`, `-b` | No       | â€”       | Bucket name to test access against (optional) |

## Examples

```bash
# Test general access
tigris credentials test

# Test access to a specific bucket
tigris credentials test --bucket my-bucket
```
