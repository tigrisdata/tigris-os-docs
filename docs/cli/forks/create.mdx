# tigris forks create

Create a new fork (copy-on-write clone) of the source bucket. Optionally fork
from a specific snapshot.

**Alias:** `c`

## Usage

```bash
tigris forks create <name> <fork-name> [flags]
t3 f c <name> <fork-name> [flags]
```

## Flags

| Name               | Required | Default | Description                          |
| ------------------ | -------- | ------- | ------------------------------------ |
| `--snapshot`, `-s` | No       | â€”       | Create fork from a specific snapshot |

## Examples

```bash
# Create a fork
tigris forks create my-bucket my-fork

# Create a fork from a specific snapshot
tigris forks create my-bucket my-fork --snapshot snap-2025-01-01
```
