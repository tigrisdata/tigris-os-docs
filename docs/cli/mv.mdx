# tigris mv

Move a folder or an object.

**Alias:** `move`

## Usage

```bash
tigris mv <src> <dest> [flags]
t3 move <src> <dest> [flags]
```

Both source and destination must be remote Tigris paths using the `t3://` (or
`tigris://`) prefix.

## Flags

| Name                | Required | Default | Description                  |
| ------------------- | -------- | ------- | ---------------------------- |
| `--recursive`, `-r` | No       | —       | Move folders recursively     |
| `--force`, `-f`     | No       | —       | Skip confirmation prompt     |

## Examples

```bash
# Move an object
tigris mv t3://my-bucket/my-path/my-object.json t3://my-bucket/new-path/my-object.json

# Move a folder recursively
tigris mv -r t3://my-bucket/my-path/ t3://my-bucket/new-path/

# Move with wildcard (skip confirmation)
tigris mv t3://my-bucket/my-path/*.jpg t3://my-bucket/new-path/ --force

# Move all contents of a bucket to another bucket
tigris mv -r t3://my-bucket/ t3://other-bucket/ --force
```
