# tigris objects delete

Delete an object.

**Alias:** `d`

## Usage

```bash
tigris objects delete <bucket> <key>
t3 o d <bucket> <key>
```

## Examples

```bash
# Delete a single object
tigris objects delete my-bucket my-file.txt

# Delete multiple objects
tigris objects delete my-bucket file1.txt,file2.txt
```
