# tigris stat

Get the details of a bucket, folder, or object.

## Usage

```bash
tigris stat <path>
t3 stat <path>
```

## Examples

```bash
# Get bucket details
tigris stat my-bucket

# Get folder details
tigris stat my-bucket/my-path

# Get object details
tigris stat my-bucket/my-path/my-object.json
```
