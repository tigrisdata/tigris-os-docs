# tigris touch

Create an empty (zero-byte) object at the given bucket/key path.

## Usage

```bash
tigris touch <path>
t3 touch <path>
```

Paths support the optional `t3://` (or `tigris://`) prefix (e.g.
`t3://my-bucket/file.txt` or just `my-bucket/file.txt`).

## Examples

```bash
# Create an empty object
tigris touch my-bucket/placeholder.txt

# Using t3:// prefix
tigris touch t3://my-bucket/logs/
```
