# Tigris Developer Documentation

<div style={{ fontSize: "20px" }}>
  Tigris is a globally distributed, multi-cloud object storage service with built-in support for the S3 API.
  It uses <strong>Dynamic Data Placement</strong> and <strong>Access-Based Rebalancing</strong> to deliver
  low-latency access worldwide — without the need to manage replication or caching.
