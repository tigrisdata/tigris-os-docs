# Partner Integration program

The Partner Integration program is designed to help partners integrate their
products with the [Tigris Object Store](https://tigrisdata.com) platform. This
program is designed to help end users use Tigris Object Store with their
preferred platform.

# Getting Started

All you need is a signing key to start integrating with Tigris Object Store. You
can get a signing key by reaching out to us at
[help@tigrisdata.com](mailto:help@tigrisdata.com). Once you have the signing
key, you can start using the API to manage resources.

Checkout the
[API reference guide](/docs/partner-integrations/api/tigris-extensions-api-reference/)
for authentication and usage.

### Connect with us

If you are interested in integrating your product with Tigris Object Store,
please reach out to us at [help@tigrisdata.com](mailto:help@tigrisdata.com)
